This repo now includes a responsive static portfolio scaffold with:

## Edit Later

- Update text in `content/site-content.js`
- Replace placeholder art in `assets/placeholders/`
- Adjust styling in `styles.css`

## Run

Open `index.html` directly in a browser, or serve the repo with any static file server.
