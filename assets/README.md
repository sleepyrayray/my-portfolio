## Assets

This scaffold uses placeholder SVGs stored in `assets/placeholders/`.

To replace them later, you can either:

1. Keep the same filenames and overwrite the placeholder files.
2. Add new image files anywhere inside `assets/` and update the `image` fields in `content/site-content.js`.

Recommended image directions:

- Card/grid thumbnails: around `1600x1200`
- Hero or spotlight stills: landscape `4:3` or `16:10`
- Keep file names lowercase and hyphenated for consistency
